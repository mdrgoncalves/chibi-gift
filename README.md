# Chibi Gift

Chibi Gift é um projeto que utiliza HTML, CSS e JavaScript para criar uma aplicação interativa. O objetivo do projeto é fornecer uma experiência divertida com mensagens personalizadas, sons e outros recursos interativos.

## Estrutura do Projeto

A estrutura do projeto é organizada da seguinte forma:

```
index.html
script.js
style.css
assets/
	font/
	img/
	music/
modules/
	messages.js
	sounds.js
```

- **index.html**: O arquivo principal que contém a estrutura HTML da aplicação.
- **script.js**: Contém a lógica principal em JavaScript.
- **style.css**: Define os estilos visuais da aplicação.
- **assets/**: Pasta que contém recursos como fontes, imagens e músicas.
  - **font/**: Fontes utilizadas no projeto.
  - **img/**: Imagens utilizadas no projeto.
  - **music/**: Arquivos de música e sons.
- **modules/**: Contém módulos JavaScript para funcionalidades específicas.
  - **messages.js**: Gerencia mensagens personalizadas.
  - **sounds.js**: Gerencia os sons da aplicação.

## Como Executar

1. Clone o repositório para sua máquina local:

   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```

2. Navegue até o diretório do projeto:

   ```bash
   cd chibi-gift
   ```

3. Abra o arquivo `index.html` em seu navegador preferido.

## Funcionalidades

- Mensagens personalizadas exibidas na tela.
- Sons interativos para melhorar a experiência do usuário.
- Design visual atrativo e responsivo.
